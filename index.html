Bss linux custom Arch linux repository

How to install:

open /etc/pacman.conf with your prefered text editor and add this line into it:

[bss-archrepo]

SigLevel = Optional TrustedOnly

Server = https://bsslinux.github.io/bss-archrepo/repo/$arch

Repo status Platforms
Active X86-64 only
